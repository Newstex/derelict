# DERELICT - IXION Clone

Space station survival RPG built with Three.js + Vite + TypeScript.

## Status

Initial development phase. Workspace exists at /home/newstex/workspace/derelict.

## Tech Stack

- Three.js
- Vite
- TypeScript
- Godot 4.6 (planned migration target)

## Directory Structure

- GameBlocks/ - Core game systems
- public/ - Static assets
- src/ - TypeScript source
- tests/ - Test suite
- DESIGN.md - Game design documentation

## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```
